# CodeAlpha-Music_Player
# Music Player using JavaScript

## Project Overview
This project is a web-based music player developed using HTML, CSS, and JavaScript.  
It allows users to play audio files, control playback,  and manage volume through an interactive and visually appealing interface.

##  Features
-  Play & Pause audio
-  Next and Previous song controls
-  Display song title and artist name
-  Show song duration and current playback time
-  Playlist support (multiple songs)

## Technologies Used
- HTML5 – Structure of the music player
- CSS3 – Styling and layout design
- JavaScript (ES6) – Audio control logic and interactivity
